# Customer Management Backend System

This project is a backend system for managing customer data and their purchase details using SQL. It allows you to add, store, and manage customer records efficiently and also export the data in CSV format.

## Features

- **Add Customer Record**: Store all relevant customer information.
- **Customer Details**:
  - **Customer Name**: Enter the full name of the customer.
  - **Gender**: Male / Female / Other.
  - **Phone Number**: Contact number of the customer.
  - **Email Address**: Example: `example@mail.com`.
  - **Customer Address**: Complete address (House No, Street, City).
- **Purchase Details**:
  - **Item Purchased**: Ring, Necklace, Bracelet, etc.
  - **Product Weight (g)**: Enter product weight in grams.
  - **ID Proof Number**: Aadhar / Passport / License.
  - **Purchase Date**: Format `dd-mm-yyyy`.
  - **Price (â‚¹)**: Enter price in rupees.
  - **Payment Method**: Cash / Other available options.
- **Export Data**: Export all customer records as CSV files for easy sharing or backup.
